Common For Loop Algorithms

Summation

Input: An array of integers.

Process: Add each value of the array to a running total.

Output: total

Mean

Input: An array of integers.

Process: Add all values together, then divide the total by the number of elements.

Output: mean value of the array

Transformation (Double)

Input: An array of integers.

Process: Create a new array where each element is twice the value of the original.

Output: new array of doubled values

Transformation (Square)

Input: An array of integers.

Process: Create a new array where each element is squared.

Output: new array of squared values

Minimum

Input: An array of integers.

Process: Compare all elements to find the smallest value.

Output: smallest number in the array

Maximum

Input: An array of integers.

Process: Compare all elements to find the largest value.

Output: largest number in the array

Count Instances (Evens)

Input: An array of integers.

Process: Count how many elements are even numbers.

Output: number of even elements

Count Instances (Positives)

Input: An array of integers.

Process: Count how many elements are greater than zero.

Output: number of positive elements

Filter Instances (Odds)

Input: An array of integers.

Process: Create a new array with only the odd numbers from the original array.

Output: new array of odd numbers

Filter Instances (Negatives)

Input: An array of integers.

Process: Create a new array containing only negative values from the original array.

Output: new array of negative numbers

Existential Quantifier (Negative Exists)

Input: An array of integers.

Process: Check if at least one value in the array is negative.

Output: true or false

Existential Quantifier (First Even)

Input: An array of integers.

Process: Scan through the array and return the first even number found.

Output: first even number or null if none exist

Universal Quantifier (All Positive)

Input: An array of integers.

Process: Check if every element in the array is positive.

Note: 0 is not a positive number.

Output: true or false

Sanity Check (All Defined)

Input: An array of any values.

Process: Verify that every element in the array is defined and not null or empty.

Output: true or false
